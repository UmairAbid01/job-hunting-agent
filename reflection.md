# Reflection

In this project, I built a file-driven Job Hunting Agent using Python.

## What I implemented:
- Reading files from multiple folders
- Extracting keywords from job descriptions and resumes
- Matching skills and calculating match score
- Generating skill-gap reports
- Providing resume improvement suggestions
- Generating interview questions from knowledge base
- Creating an application tracker (CSV file)
- Generating reminders

## Challenges:
- Understanding folder-based file handling
- Matching skills correctly
- Managing multiple outputs

## Improvements:
- Improved keyword extraction
- Organized code into functions
- Ensured all outputs are generated properly

## Future Work:
- Add GUI (Streamlit)
- Add PDF/DOCX support
- Improve AI-based suggestions
